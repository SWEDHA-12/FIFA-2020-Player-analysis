# FIFA-2020-Player-analysis
Exploratory data analysis and visualization of FIFA 2020 player dataset to uncover performance trends, player attributes, and team insights

## Problem Statement
Understand player characteristics, rating distribution, and relationships between performance attributes.

## Workflow
- Data cleaning and preprocessing
- Exploratory data analysis
- Visualization of player attributes
- Insight generation

## Key Analysis
- Player rating distribution
- Position-based performance trends
- Attribute correlations
- Top player identification

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn
